Detailed Description

Class for storing waveform lookup tables.

This is the main class in Anthem for storing and looking up values from waveforms (e.g. sine, saw, square wave). A wavetable can be constructed either by passing it a pointer to waveform values directly or through Additive Synthesis, in combination with the Partial struct. The Wavetable class implements reference counting.

Constructs a Wavetable additively.

An additive wavetable is constructed by adding a sequence of Partial objects together. Optionally they can smoothed out with sigma/lanczos approximation which also reduces the Gibbs effect.

Parameters
startAn iterator to the start of a sequence of Partial objects (e.g. std::vector<Partial>).
endAn iterator to the end of a sequence of Partial objects.
wtLengthThe length of the wavetable to construct, 4095 is used throughout Anthem.
masterAmpAttenuation value for all values in the table.
sigmaAproxBoolean whether or not to use sigma approximation to smooth out the values.
bitWidthNumber of bits to scale to, defaults to 16 (bits).
idThe wavetable's id, defaults to -1.
nameAn optional name for the Wavetable, as std::string.
