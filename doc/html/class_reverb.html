Detailed Description

A reverb-effect class.

The reverb algorithm currently implemented is the well-known and moderately popular Schroeder-Reverb, which is made up of four parallel comb filters/delay lines and two all-pass filters in series. It is a good first reverb effect but will definitely need re-implementation some time (soon).

void Reverb::setDryWet ( double  dw)
virtual

Controls the mixture between original signal and reverberated signal.

This value should usually stick around 0.1 for best sound and least distortion. Note that when the reverb becomes more wet, the input signal is attenuated proportionally, so if you have dry/wet at 0.5, the input signal will also be halved, otherwise the signal's amplitude would blow up, big time.

Parameters
dwThe dry/wet ratio (between 0 and 1).
