Detailed Description

Crossfader that ticks a crossfaded value of two GenUnits.

The Crossfader class is an extension of the CrossfadeUnit class in the sense that it has two GenUnit pointers as members. It also has a tick() method, which then returns a crossfaded value of the two GenUnit ticks.
