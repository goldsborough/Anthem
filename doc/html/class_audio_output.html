Detailed Description

Direct, real-time audio output class.

The AudioOutput class uses the RtAudio library to direct samples computed by Anthem to the OS' audio output (DAC) to output sound in real-time.

Opens a device for audio output.

Note that at initialization the default device is opened so you can output sound without calling the open() method explicitly. Use this method if you want to change the output device to something other than the default.

Parameters
idThe device ID.
channelsThe number of channels to output to (1 = mono, 2 = stereo). Defaults to and should probably stay 2.
