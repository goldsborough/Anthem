Detailed Description

Base class for any Unit that processes rather than ticks samples.

The EffectUnit class is the base class for all Units that process ticks, such as filters or effects. They differ from GenUnits in the sense that they do not produce samples on their own but only "process" existing samples and modulate them in some way such as filtering. Also, EffectUnits have a dry/wet parameter.
