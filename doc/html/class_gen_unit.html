Detailed Description

Base class for Units generating ticks or samples.

This is the base class for all Units that generate samples, such as Oscillators, Operators, Envelopes and more. Its members are a pure virtual method for generating a sample, tick(), as well as an amplitude member, amp_, along with a method to set the amplitude.
