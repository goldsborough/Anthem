Detailed Description

Bi-Quad Filter.

This is Anthem's main filter class. It is an IIR, Bi-Quad filter implemented in Direct Form II. Lots of great info can be found here: http://goo.gl/XiA8jo

References:

  • http://www.musicdsp.org/files/Audio-EQ-Cookbook.txt
  • http://www.earlevel.com/main/2013/10/13/biquad-calculator-v2/
