Detailed Description

A sequence of EnvSegs.

An EnvSegSeq is a sequence of envelope segments that are activated one after the other. Control over the individual segments is possible as well as looping.

void EnvSegSeq::setLinkedLevel ( seg_t  seg,
double  lv 
)
virtual

Sets the ending level of a segment and the starting level of the next.

This method 'links' the levels of two adjacent segments by making any segment's ending level equal to the starting level of the next. This applies to all segments except the last one, since it has no next seg.

Parameters
segThe segment in the sequence for which to set the ending amp (and starting of seg+1).
lvThe new level.
