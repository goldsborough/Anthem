Detailed Description

The Oscillator class.

An Oscillator is the most basic audio-samples-generating unit. It interpolates samples from a wavetable, which it gets from WavetableDB. It has methods to set its frequency, wavetable, phase offset and more. It is the basis for LFOs and Operators.
