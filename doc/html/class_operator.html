Detailed Description

The Operator class.

Operators are advanced Oscillators used for FM synthesis. They are the interface users ultimately interact with when producing and synthesizing music.

void Operator::setLevel ( double  level)

Sets the Operator's level, depends on the current mode.

In FM mode, the level controls the index of modulation and ranges from 0 to 10 (the index of modulation, beta, is equal to the frequency deviation of the carrier of an FM signal divided by the frequency of the modulator). In Additive mode, the level is equal to the amplitude and ranges from 0 to 1.

Parameters
modeThe new mode, 0 = FM synthesis, 1 = Additive synthesis.
Exceptions
std::invalid_argumentif the level is out of range for the current mode.
void Operator::setMode ( bool  mode)

Sets the Operator's mode, either for FM synthesis (0) or additive synthesis (1).

The mode determines the meaning of the level parameter. For FM synthesis, the level is the index of modulation and ranges from 0 to 10. For additive synthesis the level is simply the amplitude, and ranges from 0 to 1.

Parameters
modeThe new mode, 0 = FM synthesis, 1 = Additive synthesis.
