Detailed Description

An Extension to the ModEnvSegSeq for flexible segment lengths.

Neither the EnvSegSeq nor the ModEnvSegSeq classes allow for the adjustment of segments' length because some classes that inherit from them, like LFOSeq, don't let the user adjust the lengths of individual segments. This class extends ModEnvSegSeq and implements two methods for getting and setting an individual segment's length.
